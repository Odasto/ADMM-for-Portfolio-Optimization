# ADMM for Portfolio Optimization

## How to run

###Prerequisites:
- Numpy
- Scipy
- Matplotlib

### Execute

``` sh
$ python3 main.py <path to instance> [args]
```

Or

``` sh
$ ./main.py <path to instance> [args]
```

### For help execute with

``` sh
$ python3 main.py -help
```

Or

``` sh
$ ./main.py -help
```
